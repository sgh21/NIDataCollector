传感器描述NTD；按热敏电阻/非 PT100 RTD 风险分析

结论

ai0 当前读数不是有效温度信号。实测 RTD 温度在 -242.03 °C 与 1254.74 °C 两个极端值之间跳变，电阻模式在约 -499.45 Ω 与 +499.47 Ω 两个饱和值之间跳变。这不是正常温度变化，表现为量程外、开路/接线不匹配， 或传感器类型与 NI 9216 不匹配。

如果传感器确实是 NTD/NTC 热敏电阻，而不是 PT100 RTD，NI 9216 不适合直接采集它的温度。 NI 9216 是 PT100 RTD 模块，电阻量程为 0 Ω 到 400 Ω，固定 1 mA 激励；典型 NTC/NTD 热敏电阻常见阻值远高于该范围，并且需要热敏电阻公式进行非线性换算。

排查判断

可能原因 证据 处理建议
传感器不是 PT100 RTD NI 9216 对 ai0 读到电阻饱和跳变，而不是 100Ω 附近的稳定电阻。 断开传感器，用万用表量两线阻值。室温如果约 100-110Ω，才像 PT100；如果约 1kΩ、10kΩ 或更高，多半不是 NI 9216 适配对象。
NTD/NTC 热敏电阻接入了 PT100 RTD 模块 NI 9216 官方规格是 PT100、0-400Ω、1mA 激励；实测尝试 10kΩ/0.1mA 配置被驱动拒绝。 换用支持热敏电阻/高阻值电阻测量的模块，或用分压电路接普通电压输入模块，再按 Beta/Steinhart-Hart 参数软件换算。
两线 RTD 接法/DAQmx 配置不匹配 NI 9216 不接受 DAQmx TWO_WIRE 配置；官方两线 RTD workaround 要按 3-wire 任务配置。 如果确认是 PT100 两线 RTD：传感器接 RTD+ 与 RTD-，RTD- 与 COM 短接，软件选择 THREE_WIRE。
传感器开路或接线接触不良 读数稳定落在饱和值两端，符合开路/量程外表现。 断电检查端子压接；用万用表直接量传感器两端，轻晃线缆时阻值应连续稳定。

参考依据

  • NI-9216 Specifications：NI 9216 是 8 通道、0-400Ω、PT100 RTD 温度输入模块，激励电流为 1mA。
  • NI KnowledgeBase: 2-Wire RTD Support For NI 9216...：NI 9216 不正式支持两线 RTD 精度；两线 RTD workaround 是传感器接 RTD+ 和 RTD-，RTD- 与 COM 加 jumper，并按 3-wire RTD 配置。

下一步建议

  1. 先断开 ai0 传感器，用万用表测室温静态阻值，并记录加热/握住传感器时阻值是上升还是下降。
  2. 如果室温约 100-110Ω 且加热阻值上升，按 PT100 两线 RTD 处理：接 RTD+、RTD-，短接 RTD- 与 COM，软件选 THREE_WIRE。
  3. 如果室温约 10kΩ 且加热阻值下降，按 NTC/NTD 热敏电阻处理：不要用 NI 9216 直接采温，改用合适的电阻/电压采集方案和热敏电阻换算公式。
